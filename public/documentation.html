Documentation for Movie API

Business Logic URL HTTP Method Request Format Response Format
Gets a list of all movies /movies GET None A JSON object holding data about all movies.
Get data about a single movie based on title /movies/[Title] GET None A JSON object holding data about a single movie, containing description, genres, director, image URL, and whether its featured or not
Gets information about a genre based on the name /movies/genres/[Name] GET None A JSON object holding data about a single genre, containing name and description
Gets information about a direction based on the name /movies/directors/[Name] GET None A JSON object holding data about a single director, containing a bio, birth year, and death year
Creates a new user /users POST A JSON object holding the users information, containing username, email address, and password. A JSON object holding the users information, containing username, email address, and password, and a confirmation message
Updates a user's username /users/[id]/username PUT None A text confirmation message that the username has been succesfully updated
Updates a user's password /users/[id]/password PUT None A text confirmation message that the password has been succesfully updated
Updates a user's email /users/[id]/email PUT None A text confirmation message that the email has been succesfully updated
Adds a movie to a user's favorites list /users/[id]/favoritemovies POST None A text and visual confirmation message that the movie has been added to the user's favorites list
Removes a movie from a user's favorites list /users/[id]/favoritemovies DELETE None A text and visual confirmation message that the movie has been removed from the user's favorites list
Deletes a users account by their username /users/[username] DELETE None A text and visual confirmation message that the user's account has been unregistered
