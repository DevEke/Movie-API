Documentation for Movie API

Business Logic URL HTTP Method Request Format Response Format
Gets a list of all movies /movies GET None A JSON object holding data about all movies.
Get data about a single movie based on title /movies/[Title] GET None A JSON object holding data about a single movie, containing description, genres, director, image URL, and whether its featured or not
Gets information about a genre based on the name /genres/[Name] GET None A JSON object holding data about a single genre, containing name and description
Gets information about a direction based on the name /directors/[Name] GET None A JSON object holding data about a single director, containing a bio, birth year, and death year
Gets information about a specific user by their username /username/[Username] GET None A JSON object holding data about a specific user includind username, password, email, and birthday
Creates a new user /users POST A JSON object holding the users information, containing username, email address, and password. A JSON object holding the users information, containing username, email address, and password, and a confirmation message
Updates a user's username, password, or email /users/[Username] PUT A JSON object with the data the user wants to update (username, password, or email) A text confirmation message that the username, password, or email has been succesfully updated and a JSON object with the updated information
Adds a movie to a user's favorites list /users/[Username]/favorites/[MovieID] POST None A text and visual confirmation message that the movie has been added to the user's favorites list and a JSON object with the updated list of movies
Removes a movie from a user's favorites list /users/[Username]/favorites/[MovieID] DELETE None A text and visual confirmation message that the movie has been removed from the user's favorites list and a JSON object with the updated list of movies
Deletes a users account by their username /users/[Username] DELETE None A text and visual confirmation message that the user's account has been unregistered
